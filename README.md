# 🌐 Aura Galleria

A cinematic, high-end interactive website designed to showcase a futuristic mall experience through immersive visuals, smooth animations, and modern UI/UX.

---

## ✨ Features

* 🎬 Cinematic hero section with video integration
* ⚡ Smooth animations using GSAP (ScrollTrigger)
* 🧭 Scene-based scroll navigation
* 🏬 Interactive retail panels (hover expansion)
* 🗺️ Floor map with interactive hotspots
* 📊 Engaging statistics & branding sections
* 📩 Contact form with feedback overlay
* 📱 Fully responsive across devices

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript
* **Animations:** GSAP
* **Design:** Custom CSS system
* **Fonts:** Google Fonts
